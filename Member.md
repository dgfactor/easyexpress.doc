# 會員 API

+ APIPath

  例如 APIPath 為 /member 網址 API 就應該是 APIBase + APIPath => https://easy.express/{slug}/api/member

## 取得會員註冊

|請求方式||
|--------|-----|
| Method | POST |
| Url | /member/register |

### 傳入參數

Url:

**無**

Query:

**無**

Content:

| 欄位  | 型態 | 欄位說明 |
|-------|------| -------- |
| Name | string | 會員帳號 |
| Email | email | 電子郵件 |
| Password | string | 密碼 |

### 回傳結果
| 欄位  | 型態 | 欄位說明 |
|-------|------|----------|
| Id | integer | 會員 ID |

### 錯誤
| 狀態碼  | 錯誤說明 |
|---------|----------|
| 400 | 錯誤的請求 |

***
